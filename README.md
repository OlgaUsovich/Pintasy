<!-- GETTING STARTED -->
### Launch the project

1. Clone the repo
   ```sh
   git clone https://github.com/OlgaUsovich/Pintasy.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```sh
   npm run dev
   ```
